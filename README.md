# Compose Tutorials

This repository contains a collection of small tutorials and examples built with **Jetpack Compose**.  
It is intended as a learning and experimentation space for modern Android UI development using Kotlin.

## Purpose

- Explore core Jetpack Compose concepts
- Practice building UI components declaratively
- Serve as a reference for common Compose patterns

## Structure

Each folder or module focuses on a specific topic or tutorial, keeping examples small and self-contained.

## Requirements

- Android Studio (latest stable recommended)
- Kotlin
- Basic knowledge of Android development

## Usage

Clone the repository and open it in Android Studio.  
You can run individual modules or examples depending on the project structure.

## Status

This repository is a work in progress and may be updated with additional tutorials over time.
